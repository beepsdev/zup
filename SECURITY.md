# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in Zup, please report it responsibly.

**Do not open a public issue.** Instead, email **security@beeps.dev** with:

- A description of the vulnerability
- Steps to reproduce
- Any relevant logs or screenshots

We'll acknowledge your report within 48 hours and work with you on a fix before any public disclosure.

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.x     | Yes       |
