# zupdev

Open source reliability agent that continuously runs the OODA loop (Observe, Orient, Decide, Act) on your production systems.

> **Bun-only** — this package ships `.ts` source and requires [Bun](https://bun.sh) as the runtime.

## Install

```bash
bun add zupdev
```

## Documentation

Visit [zup.dev](https://zup.dev) for full documentation, guides, and examples.
