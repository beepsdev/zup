# Zup

Zup is an open source reliability agent that runs the OODA loop (Observe, Orient, Decide, Act) on your production systems. It observes signals across your stack, builds situational awareness from current state and historical context, decides on the safest response, and acts automatically or with human approval.

## What you can do

- Monitor endpoints, deployments, and infrastructure signals.
- Correlate incidents with recent changes.
- Run safe remediation actions with approvals.
- Extend everything through plugins.
